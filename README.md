# Starter project with Vite (vue.js 3.x) & Nest.js (8.x)
This is a starter project to speed up the implementation of new projects under Vue.js, Nest.js and Mongodb.

## Additional packages
- Tailwind css 3.x

## Installation
```shell
yarn
```

## Run api
```shell
yarn run dev:api
```

## Run client
```shell
yarn run dev:client
```
